AwesomeSoundLib

aweslib

AwesomeSoundLib (aweslib) is a site for a code library on the minecraft server DiamondFire. If you want to use this library, grab the code from the Code Vault (Plot 43780) by searching for AwesomeSoundLib via an @ command (@[search term] ). To generate a tune, press on the "New" button on this website, choose an instrument and set a pitch from 0 to 2 (supports decimals). When you have got a tune, press "Generate" to get a string with the code of your tune, which will be your song. To use this song in DiamondFire, create a new function named something like "exampleSong" and inside that function create a local List variable named "aweslib:notes" and inside put your string generated via the tune generator above inside that list. Once you have done that, if you want to play it in DF then call your previous function (our example was called "exampleSong") then call the function "aweslib:song".
